# Star Wars Character 

_A Frontend Take-Home Assignment by **Amit Kumar**_

---

## Overview

A responsive **Star Wars Character Explorer** built using **React + TypeScript + Tailwind CSS**, which fetches data from the [Star Wars API (SWAPI)](https://swapi.dev/).

This project demonstrates frontend development skills in **UI design**, **API integration**, **state management**, and **performance optimization** using modern React tools.

---
## How to Run Locally

### Clone the repository

git clone https://github.com/AmitKumar2k2/zippee-assessment

### Install dependencies
- npm install
- npm install -D tailwindcss@3
- npx tailwindcss init  

npm run dev
